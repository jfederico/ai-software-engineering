# Support

## Where to Ask Questions

- Usage questions: GitHub Discussions (recommended)
- Bug reports: GitHub Issues
- Security concerns: See SECURITY.md and report privately

## What to Include

For faster help, include:
- Which methodology you are using
- Which file or prompt step you are on
- Expected behavior vs actual behavior
- Minimal reproducible context

## Response Expectations

This is an open-source project with best-effort support. Community contributions and peer answers are welcome.
